## Introduction
Guide to solve a rubik cube by walking through each layers
- First 2 Layer F2L
  - 1st Layer (guide)
  - 2nd Layer (guide)
- Last Layer
  - Orientation of Last Layer OLL (calculator)
  - Permutation of Last Layer PLL (calculator)
